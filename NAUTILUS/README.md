# \# Nautilus Code Samples

# 

# These are selected C# scripts from \*Nautilus\*, a first-person underwater exploration game made in Unity.

# 

# My work on the project included level design, audio implementation, gameplay support, and small scripting tasks used to improve feedback, atmosphere, and player readability.

# 

# \## Folders

# 

# \### ScoutMechanic

# 

# Scripts connected to the scout tool mechanic, including projectile behavior, light feedback, and distance-based visual feedback.

# 

# \### GameJuice

# 

# Small scripts used to add atmosphere and interaction polish, such as flickering lights, moving needles, random steam bursts, and small interactive details.

# 

# \## Note

# 

# These scripts are presented as portfolio code samples. They are cleaned-up examples and are not meant to represent the full project source.
