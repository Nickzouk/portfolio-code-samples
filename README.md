# Portfolio Code Samples

This repository contains a small collection of cleaned-up code samples from my game projects.

The goal is not to show full project source code, but to highlight specific systems, tools, and player-facing features I worked on during production.

## Projects

### Nautilus

Code samples from *Nautilus*, a first-person underwater exploration game made in Unity.

These scripts focus on small gameplay and feedback systems, including the scout mechanic, visual feedback, interaction polish, and atmospheric game juice.

## Note

These samples have been cleaned up for portfolio use and may not include full project dependencies, assets, or scene setup.
